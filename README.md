## Installer et lancer

Pour utiliser cette application vous devez:
1. Clonez le repo sur votre machine:
```bash
git clone https://<username>:<PAT>@git.iut-orsay.fr/edagos1/s2-sae-dev-app-ef1.git
```

2. Ouvrez le projet `app-gestion-ciup` dans eclipse. Et lancez l'une des deux configurations de lancement suivante:
    1. `Run main app` pour lancer l'application.
    2. `Run tests` pour lancer les tests.
